# Crosspose.Core

Shared helpers for the Crosspose toolchain.

## Included
- `Diagnostics/ProcessRunner`: Captures stdout/stderr/exit codes from external tools (helm, docker, podman).
- `Logging/CrossposeLoggerFactory`: Consistent console logging across CLIs and the future GUI host.
- Orchestration and source helpers shared by `Crosspose.Cli` and future GUI layers.
